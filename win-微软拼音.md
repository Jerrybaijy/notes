# Overview

**微软拼音（Microsoft Pinyin IME）**是微软公司开发的 Windows 系统级简体中文输入法。它最大的特点是**“干净、稳定、深度集成”**，无需额外安装，是许多追求系统原生体验用户的首选。

> [Microsoft 简体中文 IME](https://support.microsoft.com/zh-cn/windows/microsoft简体中文-ime-9b962a3b-2fa4-4f37-811c-b1886320dd72#id0ebh=microsoft_pinyin)

# Install

微软拼音是 Windows 系统的**内置组件**，通常无需单独下载安装包。

1. **打开设置**： `设置` > `时间和语言` > `语言和区域`
2. **添加中文语言**： 在 `首选语言` 列表中 添加"简体中文"。
3. **启用微软拼音**： 点击 `简体中文` 语言项下的 `语言选项`。 在 `键盘` 区域，点击 `添加键盘`，从列表中选择 **【微软拼音】**。
4. **切换使用**：添加后，你可以按键盘上的 `Win + 空格键` 快速切换到微软拼音输入法。

# 特殊模式

## V 模式

**V 模式**用于对数字的特殊处理，如输入中文格式的数字、日期、时间、公式等。

## U 模式

**V 模式**用于**拆字**和**特殊符号**。

# 亮点

## 自定义短语

- `输入法设置` > `词库和自学习` > `用户定义的短语` > `添加`
- **实战**：将邮箱设置为短代码。
  - 拼音：`yx`
  - 短语：`your-email@example.com`
- **效果**：中文状态下输入 `yx` 直接将 `your-email@example.com` 显示在候选区。

## 音节切分

输入“西安”时，打 `xi'an`（加单引号），能精准切分，避免打成“先” 。

## 以词定字

先输入一个包含目标字的词语，再利用快捷键从这个词语中“抠”出你需要的那个字。这个功能专门用来解决**生僻字、候选位靠后的字**输入困难的问题。

- `输入法设置` > `按键` > `以词定字`
- **痛点**：想打“**逍遥**”的“**逍**”字，输入 `xiao` 可能要翻好几页。
- **以词定字解法**：输入 `xiaoyao`（逍遥）→ 按快捷键取首字 → 直接上屏“逍”。

# 不足

- 开启 `Caps Lock` 以后，语言只显示“中”或“英”，不显示大写时的 A。
  - 无法通过 `Shift` 键切换到小写状态。
  - 无法通过连按 `Shift` 键切换到中文状态。
