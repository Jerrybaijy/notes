# Overview

[**十字军东征**](https://zh.wikipedia.org/wiki/十字军东征)（1096年—1291年）是一系列在天主教教宗的准许下，对[伊斯兰政权](https://zh.wikipedia.org/wiki/伊斯兰政权)发动的持续近 200 年的战争。参加这场战争的士兵配有十字标志，因此称为**十字军**。

十字军的最初目的是收复被穆斯林统治的圣地耶路撒冷，但随着事态发展，战争性质发生了改变。

# 第一次十字军东征

[**第一次十字军东征**](https://zh.wikipedia.org/wiki/第一次十字军东征)（1096年—1099年）旨在收复穆斯林的征服中被占领的黎凡特圣地，最终以 1099 年十字军攻陷耶路撒冷收尾，并建立了十字军国家的[耶路撒冷王国](https://zh.wikipedia.org/wiki/耶路撒冷王国)和三个附属小国：[伊德萨伯国](https://zh.wikipedia.org/wiki/伊德萨伯国)、[的黎波里伯国](https://zh.wikipedia.org/wiki/的黎波里伯国)、[安条克公国](https://zh.wikipedia.org/wiki/安条克公国)。

# 第四次十字军东征

[**第四次十字军东征**](https://zh.wikipedia.org/wiki/第四次十字军东征)（1202年—1204年）目的本来是要进攻[阿尤布王朝](https://zh.wikipedia.org/wiki/阿尤布王朝)所在的[埃及](https://zh.wikipedia.org/wiki/埃及)，收复被穆斯林控制的圣城耶路撒冷。然而一系列经济和政治事件最终导致十字军军队攻陷君士坦丁堡。

# 第五次十字军东征

[**第五次十字军东征**](https://zh.wikipedia.org/wiki/第五次十字军东征)（1217年—1221年）是受教宗所支配的最后一次十字军，以埃及[阿尤布王朝](https://zh.wikipedia.org/wiki/阿尤布王朝)为进攻目标。

几乎于此同时，1219年至1221年，蒙古帝国成吉思汗发动[蒙古第一次西征](https://zh.wikipedia.org/wiki/蒙古第一次西征)，灭穆斯林的[花剌子模帝国](https://zh.wikipedia.org/wiki/花剌子模帝國)，令伊斯兰世界两面受敌。

# 第九次十字军东征

[**第九次十字军东征**](https://zh.wikipedia.org/wiki/https://zh.wikipedia.org/wiki/)（1271年—1272年）有时也被合并成第八次东征的一部分。

1291年，最后一个十字军据点[阿卡](<https://zh.wikipedia.org/wiki/阿卡_(以色列)>)被埃及[马木留克](https://zh.wikipedia.org/wiki/马木留克)铁骑攻陷，[耶路撒冷王国](https://zh.wikipedia.org/wiki/耶路撒冷王国)灭亡。
