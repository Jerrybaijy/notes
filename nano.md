# Overview

**nano** 是 Linux 和类 Unix 系统中最易上手的**命令行文本编辑器**。

```bash
nano <file>
```

| 操作     | 快捷键     | 说明                            |
| -------- | ---------- | ------------------------------- |
| **保存** | `Ctrl + O` | 写入文件，回车确认文件名        |
| **退出** | `Ctrl + X` | 退出 nano，若有未保存更改会提示 |
| **搜索** | `Ctrl + W` | 查找文本，支持简单正则          |
