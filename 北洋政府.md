# 北洋

[**北洋**](https://zh.wikipedia.org/wiki/北洋)，原指中国华北靠海一带的疆域，一般是指渤海、黄海，朝鲜半岛四周，如奉天省（今辽宁省）、直隶省（今河北省、北京市、天津市）、山东省。

由“北洋”一词延伸出许多关于“北洋”的词汇，如“[北洋大臣](https://zh.wikipedia.org/wiki/北洋大臣)”、“[北洋水师](https://zh.wikipedia.org/wiki/北洋水师)”、“[北洋政府](https://zh.wikipedia.org/wiki/北洋政府)”、“[北洋军阀](https://zh.wikipedia.org/wiki/北洋军阀)”、“[北洋武备学堂](https://zh.wikipedia.org/wiki/北洋武備學堂)”、“[北洋大学](https://zh.wikipedia.org/wiki/北洋大学)”、“[北洋拳术馆](https://zh.wikipedia.org/wiki/北洋拳術館)”。这些词均源自李鸿章担任北洋大臣以后，李鸿章之后，下一个北洋派的著名领袖为袁世凯。

# 北洋政府

[**北洋政府**](https://zh.wikipedia.org/wiki/北洋政府)，又称**北京政府**，正式名称为**中华民国政府**，指中华民国建国初期，以京兆地方为首都的中央政府。因为大总统和国务总理主要由[北洋军阀](https://zh.wikipedia.org/wiki/北洋军阀)出身，被南京国民政府贬抑为“北洋政府”。

1912年2月12日，清朝[隆裕太后](https://zh.wikipedia.org/wiki/隆裕太后)诏授[袁世凯](https://zh.wikipedia.org/wiki/袁世凯)，在北京全权组建临时政府。2月13日，[孙中山](https://zh.wikipedia.org/wiki/孙中山)遵守[南北议和](https://zh.wikipedia.org/wiki/南北议和)时的承诺，向参议院辞职，并推举北洋军首领[袁世凯](https://zh.wikipedia.org/wiki/袁世凯)为大总统。2月15日，袁世凯被选为[第二任临时大总统](https://zh.wikipedia.org/wiki/1912年中华民国临时大总统选举)。

1913年10月6日，袁世凯当选为首任正式中华民国大总统，象征着北京政府正式成立，中华民国临时政府结束。

袁世凯未曾一统南方的[革命党](https://zh.wikipedia.org/wiki/中華革命黨)势力，袁死后，无人有声望和能力统御整个[北洋军](https://zh.wikipedia.org/wiki/北洋軍)，北洋军遂分裂为[直系](https://zh.wikipedia.org/wiki/直系軍閥)、[皖系](https://zh.wikipedia.org/wiki/皖系軍閥)、[奉系](https://zh.wikipedia.org/wiki/奉系軍閥)等三大割据势力，统称[北洋军阀](https://zh.wikipedia.org/wiki/北洋軍閥)。

1926年7月，广州国民政府发起[北伐](https://zh.wikipedia.org/wiki/國民革命軍北伐)，北洋军战败后退据东北。北洋政府最后一任国家元首、陆海军大元帅[张作霖](https://zh.wikipedia.org/wiki/张作霖)撤退途中遭日本军队炸死，其子[张学良](https://zh.wikipedia.org/wiki/张学良)在继承政权后于1928年12月29日宣布[东北易帜](https://zh.wikipedia.org/wiki/东北易帜)，服从中华民国国民政府统治，北洋政府正式宣告结束。
