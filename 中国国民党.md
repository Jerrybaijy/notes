# 概述

[**中国国民党**](https://zh.wikipedia.org/wiki/中国国民党)（英语译名：Kuomintang  或 Chinese Nationalist Party），简称**国民党**（KMT），是中华民国主要政党之一。

其前身为1894年11月24日成立于夏威夷檀香山[兴中会](https://zh.wikipedia.org/wiki/兴中会)，后先后改组为[中国同盟会](https://zh.wikipedia.org/wiki/中国同盟会)、[国民党](https://zh.wikipedia.org/wiki/國民黨_(1912年))、[中华革命党](https://zh.wikipedia.org/wiki/中華革命黨)，1919年10月10日经[孙中山](https://zh.wikipedia.org/wiki/孫中山)于上海法租界改组而改用现名，1949年随中华民国政府迁台而从中国大陆转以台湾为基地。

# 历史沿革

- 【**兴中会**】1894年11月24日，[孙中山](https://zh.wikipedia.org/wiki/孙中山)于夏威夷檀香山成立[**兴中会**](https://zh.wikipedia.org/wiki/兴中会)。
    - 中国国民党视其为奠基并因此以其成立日为建党日。
- 【**中国同盟会**】1905年8月20日，由[孙中山](https://zh.wikipedia.org/wiki/孫中山)、[宋教仁](https://zh.wikipedia.org/wiki/宋教仁)、[黄兴](https://zh.wikipedia.org/wiki/黄兴)、[黄元秀](https://zh.wikipedia.org/wiki/黄元秀)、[蔡元培](https://zh.wikipedia.org/wiki/蔡元培)等人领导，兴中会与[华兴会](https://zh.wikipedia.org/wiki/華興會)、[光复会](https://zh.wikipedia.org/wiki/光復會)和其他各会合并为[**中国同盟会**](https://zh.wikipedia.org/wiki/中国同盟会)。
- 【**国民党**】1912年8月25日，由宋教仁主导，中国同盟会与其他党派合并为[**国民党**](https://zh.wikipedia.org/wiki/国民党)。
    - 1913年11月4日被袁世凯解散
- 【**中华革命党**】1914年7月8日，孙中山在日本东京创立[**中华革命党**](https://zh.wikipedia.org/wiki/中华革命党)，党员多为原[兴中会](https://zh.wikipedia.org/wiki/興中會)、[中国同盟会](https://zh.wikipedia.org/wiki/中國同盟會)及1912年成立的[国民党](https://zh.wikipedia.org/wiki/國民黨_(1912年))的部分人士。
- 【**中国国民党**】1919年10月10日，由孙中山主导，将中华革命党改名为[**中国国民党**](https://zh.wikipedia.org/wiki/中国国民党)。
