# 概述

[**中华民国**](https://zh.wikipedia.org/wiki/中华民国)是位于东亚的民主共和国，曾在国际上广泛代表[中国](https://zh.wikipedia.org/wiki/中國)，现今多通称“[台湾](https://zh.wikipedia.org/wiki/臺灣)”。

# 历史沿革

- 【**辛亥革命** · **武昌起义**】1911年10月10日，[中国同盟会](https://zh.wikipedia.org/wiki/中国同盟会)领导革命军发动[武昌起义](https://zh.wikipedia.org/wiki/武昌起義)，各省宣布脱离清朝统治自行独立，是为[辛亥革命](https://zh.wikipedia.org/wiki/辛亥革命)。
- 【**南京临时政府**】1912年1月1日，各省都督府代表联合会在南京成立[中华民国临时政府](https://zh.wikipedia.org/wiki/中華民國臨時政府_(1912年—1913年))，即**南京临时政府**，选举孙中山为[临时大总统](https://zh.wikipedia.org/wiki/中華民國臨時大總統)。
    - 后临时政府为纪念革命军武昌起义成功，遂定每年10月10日为[中华民国国庆日](https://zh.wikipedia.org/wiki/國慶日_(中華民國))，称“双十节”。
- 清朝内阁总理大臣[袁世凯](https://zh.wikipedia.org/wiki/袁世凱)和中华民国临时政府[南北议和](https://zh.wikipedia.org/wiki/南北議和)，逼迫[宣统退位](https://zh.wikipedia.org/wiki/宣統退位)；孙中山则主动辞职，由袁世凯继任临时大总统。
- 【**北京临时政府**】1912年3月10日，[袁世凯](https://zh.wikipedia.org/wiki/袁世凯)于北京就任临时大总统职务，即**北京临时政府**。
- 【**北洋政府**】1913年10月10日，袁世凯被选举为[中华民国正式大总统](https://zh.wikipedia.org/wiki/中華民國大總統)，[北洋政府](北洋政府.md)成立，其后[内阁](https://zh.wikipedia.org/wiki/中华民国北京政府内阁列表)由[北洋军阀](https://zh.wikipedia.org/wiki/北洋軍閥)掌控，直至1928年12月29日宣布[东北易帜](https://zh.wikipedia.org/wiki/东北易帜)。
- 【**护法军政府**】[护法运动](https://zh.wikipedia.org/wiki/護法運動)期间，1917年9月1日，孙中山主导成立[护法军政府](https://zh.wikipedia.org/wiki/)，对抗北洋政府。
- 【**广州中华民国政府**】1921年4月2日，孙中山建立[广州中华民国政府](https://zh.wikipedia.org/wiki/广州中华民国政府)，继承自护法军政府。
- 【**中华民国陆海军大元帅大本营**】1923年3月2日，孙中山在广州创建[中华民国陆海军大元帅大本营](https://zh.wikipedia.org/wiki/中华民国陆海军大元帅大本营)
    - 中国大陆历史教科书亦称之为**广东革命政府**
    - 在1924年正式改用[青天白日满地红旗](https://zh.wikipedia.org/wiki/青天白日满地红旗)为国旗
    - 孙中山逝世后，[汪精卫](https://zh.wikipedia.org/wiki/汪精卫)于1925年7月1日改组为[国民政府](https://zh.wikipedia.org/wiki/國民政府)，任[国民政府主席](https://zh.wikipedia.org/wiki/國民政府主席)。
- 【**国民政府**】
    - 【**广州**】1925年3月12日[孙中山](https://zh.wikipedia.org/wiki/孫中山)逝世后，原[陆海军大元帅大本营](https://zh.wikipedia.org/wiki/中华民国陆海军大元帅大本营)改组为[**国民政府**](https://zh.wikipedia.org/wiki/國民政府)，以[汪精卫](https://zh.wikipedia.org/wiki/汪精卫)为首任主席；此时，国民政府通称“广东革命政府”，割据于广州，与位于北京的[北洋政府](北洋政府.md)相互对峙。
    - 【**武汉**】[国民革命军北伐](https://zh.wikipedia.org/wiki/國民革命軍北伐)期间，国民政府的行政中心亦一路北上迁移至武汉、南京等地。1927年1月，随着[北伐战争](https://zh.wikipedia.org/wiki/北伐戰爭)胜利，国民政府迁至武汉，称“武汉国民政府”。
    - 【**南京**】1927年4月18日，蒋介石遂在南京另组“**南京国民政府**”，与武汉国民政府分庭抗礼，即[宁汉分裂](https://zh.wikipedia.org/wiki/寧漢分裂)。
    - 1927年8月，武汉国民政府和南京国民政府合并，即[宁汉合流](https://zh.wikipedia.org/wiki/宁汉合流)。
    - 【**洛阳**】1932年1月28日，[一·二八事变](https://zh.wikipedia.org/wiki/一·二八事變)，国民政府迁都洛阳作为临时首都。12月1日，国民政府才迁回南京。
    - 【**重庆**】1937年[七七事变](https://zh.wikipedia.org/wiki/七七事變)后，1937年11月17日，国民政府迁至武汉，11月21日正式迁至重庆办公。12月1日，国民政府在重庆开始办公。
    - 【**南京**】1945年，中国抗日战争随着[第二次世界大战](https://zh.wikipedia.org/wiki/第二次世界大战)结束。1946年5月5日，国民政府还都南京。
    - 【**中华民国政府**】1948年5月20日，[蒋中正](https://zh.wikipedia.org/wiki/蔣中正)与[李宗仁](https://zh.wikipedia.org/wiki/李宗仁)于南京总统府宣誓就任第一任[中华民国总统](https://zh.wikipedia.org/wiki/中華民國總統)与[中华民国副总统](https://zh.wikipedia.org/wiki/中華民國副總統)，国民政府改组为[中华民国政府](https://zh.wikipedia.org/wiki/中華民國政府)。
