# Overview

[**ClawHub**](https://clawhub.ai/) 是 OpenClaw 的官方 skill 仓库。

# Install

```bash
npm install -g clawhub
clawhub -V
```

# 其它

## 安装 Skill

```bash
# 安装 skill
clawhub install <skill-slug>
```

通过 `clawhub` 命令安装的 skill 会安装在工作区目录。
