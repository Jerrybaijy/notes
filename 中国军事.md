# 概述

[**中国共产党和中华人民共和国中央军事委员会**](https://zh.wikipedia.org/wiki/中央军事委员会)，通称**中央军事委员会**，简称**中央军委**、**军委**，是[中国共产党](https://zh.wikipedia.org/wiki/中国共产党)和[中华人民共和国](https://zh.wikipedia.org/wiki/中华人民共和国)[武装力量](https://zh.wikipedia.org/wiki/武装力量)最高军事领导机构的总称。一个机构两块牌子：

- **中国共产党中央军事委员会**（简称“中共中央军委”）
- **中华人民共和国中央军事委员会**（简称“国家中央军委”）

[**中华人民共和国武装力量**](https://zh.wikipedia.org/wiki/中华人民共和国军事)由[中国人民解放军](https://zh.wikipedia.org/wiki/中國人民解放軍)、[中国人民武装警察部队](https://zh.wikipedia.org/wiki/中國人民武裝警察部隊)和[中国民兵](https://zh.wikipedia.org/wiki/中国民兵)组成。
