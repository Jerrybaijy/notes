# 概述

[**中华民国政府**](https://zh.wikipedia.org/wiki/中华民国政府)是[中华民国](https://zh.wikipedia.org/wiki/中華民國)的治权机构，其历史最早可追溯至1911年肇建于武昌的[中华民国军政府](https://zh.wikipedia.org/wiki/中華民國軍政府鄂軍都督府)，之后历经[南京临时政府](https://zh.wikipedia.org/wiki/中華民國臨時政府_(南京))、[北京临时政府](https://zh.wikipedia.org/wiki/中华民国临时政府_(1912–1913))、[北洋政府](https://zh.wikipedia.org/wiki/北洋政府)、[国民政府](https://zh.wikipedia.org/wiki/國民政府)等多次政权替换。
