# 单位建制

## 单位行政级别

| 行政级别 |                             单位                             |
| :------: | :----------------------------------------------------------: |
|  国家级  | 中共中央<br />国务院<br />全国人大<br />全国政协<br />中央军委 |
|  副国级  |              政治局常委<br />最高法<br />最高检              |
|  省部级  |             省、自治区、直辖市<br />国务院各部委             |
|  副省级  |                      副省级城市（15个）                      |
|  厅局级  |              地级市、自治州、地区、盟<br />省厅              |
|  县处级  |               县、县级市、市辖区<br />县公安局               |
|  乡科级  |                         乡、镇、街道                         |
|  股所级  |                            派出所                            |

建制单位行政级别差值一般为 1 格行政级别，如长春市长原为副省级，市局局长原为副厅级。

**副省级城市**的建制级别比常规地级市高半格，包含：

- **10 个省会城市**：沈阳、长春、哈尔滨、南京、杭州、济南、广州、武汉、成都、西安
- **5 个计划单列市**：大连、青岛、宁波、深圳、厦门
- 并非所有省会都是副省级城市。

## 单位主官级别

正常情况，单位主官的行政级别与单位级别相同。但对于具有特殊地位的单位，其主官的行政级别往往存在干部高配的情况，一般比单位行政级别高半格。
