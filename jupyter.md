# Jupyter

Jupyter 是一个开源项目，用于支持跨语言的交互式数据科学和科学计算。其核心是 Jupyter Notebook，提供了一个结合代码、输出、文档和多媒体的 Web 环境。

Jupyter 生态核心：

- **Jupyter Notebook**：原始的、基于 Web 的交互式计算环境。
- **JupyterLab**：新一代的 Jupyter Web 界面，提供更灵活、模块化的 IDE 体验（包括文件浏览器、终端、代码编辑器）。
- **Notebook 文件**：以 `.ipynb` 为扩展名的 JSON 文件，存储代码、输出和 Markdown 文本。
